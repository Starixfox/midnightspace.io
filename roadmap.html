Indicative plan

Roadmap

An honest, indicative plan for what Midnight Space is working on and where it is heading. Timelines are approximate and subject to change based on feedback and priorities.

This roadmap is intended to give a realistic picture of progress and direction. It is not a commitment or a product announcement. Dates are approximate.

Development timeline

Now — Current

Prototype phase

The Saudi Space Opportunity Hub is live as a working prototype. Data is manually curated. The dashboard is functional and has been demonstrated to prospective users.

  • Saudi Space Opportunity Hub demo — live and demonstrable
  • Manually curated database of Saudi, GCC, and global space programs
  • Filterable list and basic chart views
  • Dashboard designed for government and agency decision-makers
  • Actively seeking pilot partners and early feedback
Active now
~1–2 months — Target

First product version

Consolidate the prototype into a usable initial product with a proper web interface, better data organisation, and the core filtering and viewing experience cleaned up for external use.

  • Consolidated web dashboard with improved navigation
  • Better filters — status, region, sector, program type, funding range
  • Improved charts and summary views for briefings
  • Basic export capability — list downloads or shareable views
  • Early access onboarding for first pilot partners
Target: ~1–2 months
~2 months — Goal

First paying customer

The target is to have the first organisation using the platform with a commercial arrangement in place — whether a direct subscription, a pilot fee, or a service agreement.

  • First paying organisation using the platform in a real workflow
  • Understanding validated through actual usage — not just demos
  • Pricing model tested and iterated based on pilot feedback
  • At least one additional partner identified for the pipeline
Goal: ~2 months
Later — Direction

Broader coverage and API access

Expand the data platform beyond the initial GCC focus to include major space funding programs from Europe, North America, and Asia. Add API access for organisations that want to pull data into their own tools.

  • Expand database to cover global space programs and funding bodies
  • Add more data sources — international agencies, bilateral programs
  • Structured API for integration into internal tools and reports
  • Alerts and notification system for new programs and deadline changes
  • Custom views configurable per-organisation
Planned direction
Longer term — Vision

Ecosystem intelligence tools

The longer-term direction is deeper analytics — moving from program lookup toward genuine intelligence about space ecosystem dynamics, trends, and strategic scenarios.

  • Trend analysis — how funding patterns shift over time
  • Scenario views for strategic planning and policy modelling
  • Comparative intelligence across countries and agencies
  • Integration with commercial data sources (funding rounds, partnerships)
  • Tools for ecosystem mapping and gap analysis at a national level
Longer-term vision
